## 🎮게임 설명
주인공이 악몽에서 벗어나기 위해 악몽을 이겨내는 내용의 게임입니다. 꿈속에서처럼 맵이나 진행 방향이 갑자기 바뀌는 기능을 게임으로 만들고 싶어서 제작한 게임으로 게임 속에 맵의 배경이 바뀌거나 위아래가 뒤집히는 등 **다양한 맵 트릭**을 구현했습니다. 게임은 총 **세 개의 스테이지**로 구성되어 있고, 일반 맵과 보스전으로 이루어져 있습니다.
